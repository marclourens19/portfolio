I'm Joshua Marc Lourens, a software development student based in South Africa, focused on becoming a junior developer who ships polished, dependable software.

★ Featured

Cybersecurity Awareness Chatbot

WPF Desktop App · NLP · MySQL

A conversational desktop assistant that teaches cybersecurity. It runs a knowledge base of 13 topics with randomized responses, fuzzy matching for typo tolerance, and sentiment detection that adapts its tone. Layered on top: a MySQL-backed task manager with reminders, a 13-question interactive quiz, a timestamped activity log, and a polished GUI with voice greetings and typewriter text. Built with a clean engine / data / UI separation and graceful offline handling.

  • C#
  • .NET 10
  • WPF / XAML
  • MySQL
  • NLP
  • Fuzzy keyword matching & intent detection
  • Sentiment-aware, empathetic responses
  • Persistent tasks + reminders in MySQL
★ Featured

EventEase — Cloud Event & Venue Booking

ASP.NET Core MVC · Microsoft Azure

A full event-management platform built to be cloud-native. Admins create venues (with image uploads), schedule events, and take bookings — backed by conflict detection that prevents double-booking the same venue. It evolved across three phases from a local database to fully hosted Azure infrastructure: Azure SQL, Blob Storage for images, App Service hosting, and a CI/CD workflow.

  • ASP.NET Core MVC
  • C#
  • EF Core
  • Azure SQL
  • Blob Storage
  • Bootstrap 5
  • Booking conflict detection (no double-bookings)
  • Azure SQL + Blob Storage + App Service
  • CI/CD pipeline via GitHub Actions

NipponQuest — Gamified Japanese Learning

Full-Stack Web App · Capstone · YugenLabs

A web platform that turns learning Hiragana & Katakana into an RPG-style quest. Built full-stack with secure user accounts, it layers real game mechanics over study: a custom XP & leveling system, a global leaderboard, daily streaks to build habits, and real-time validation on character practice — all on a mobile-first interface. My capstone project, built to enterprise architecture standards.

  • SQL Server
  • ASP.NET Identity
  • Bootstrap
  • XP, levels & persistent progression
  • Global leaderboard + daily streak system
  • Secure auth via ASP.NET Core Identity
